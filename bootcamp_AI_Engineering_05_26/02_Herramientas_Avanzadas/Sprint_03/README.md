# 📘 Sprint 03 — Herramientas Avanzadas para AI Engineering

## Bienvenido al Sprint 03

En este sprint dominarás las herramientas fundamentales para el procesamiento de datos en AI Engineering. Aprenderás a trabajar con NumPy para computación numérica eficiente y Pandas para manipulación de datos a escala, habilidades críticas para construir modelos de IA robustos.

Este sprint se divide en **dos unidades temáticas**, cada una con su propia sesión de aprendizaje, ejercicios y práctica obligatoria.

---


## 🎯 Contenidos del Sprint

### 📖 Unidad 01 — NumPy

**Tema Central**: Computación numérica eficiente con NumPy

NumPy es la biblioteca fundamental para computación científica en Python. En esta unidad aprenderás a crear y manipular arrays multidimensionales, realizar operaciones matemáticas complejas y optimizar el rendimiento de tus cálculos.

**¿Qué aprenderás?**

- Arrays de NumPy: creación, indexación y slicing
- Operaciones vectorizadas vs. bucles en Python
- Álgebra lineal: matrices y operaciones matriciales
- Funciones matemáticas y estadísticas
- Broadcasting: operaciones con arrays de diferente forma
- Rendimiento y optimización con NumPy
- Aplicaciones en Machine Learning y procesamiento de datos

**Estructura de la Unidad**:
- 📚 **Teoría** — Material de lectura con ejemplos
- 🎯 **Workout** — Ejercicios guiados para practicar
- 💪 **Ejercicios** — Retos independientes
- 🏆 **Práctica Obligatoria** — Proyecto para evaluar

**Live Review 1**: Lunes 15 de Junio, 19:00–21:00

### 📊 Unidad 02 — Pandas: Lectura, Escritura y Manipulación de Ficheros

**Tema Central**: Manipulación de datos con DataFrames

Pandas es la herramienta estándar para procesamiento de datos en Python y AI Engineering. En esta unidad aprenderás a leer, escribir, limpiar y transformar datos en múltiples formatos, habilidades indispensables para preparar datasets para modelos de IA.

**¿Qué aprenderás?**

- DataFrames y Series: estructura y manipulación
- Lectura de ficheros: CSV, Excel, JSON, SQL
- Escritura de datos en diversos formatos
- Limpieza de datos: valores nulos, duplicados, outliers
- Selección y filtrado de datos
- Agrupación y agregación de datos
- Merging y concatenación de DataFrames
- Exploración y validación de datos para modelos de IA

**Estructura de la Unidad**:
- 📚 **Teoría** — Notebooks con conceptos y ejemplos
- 🎯 **Workout** — Ejercicios guiados en Colab/VSCode
- 💪 **Ejercicios** — Problemas de manipulación de datos
- 🏆 **Práctica Obligatoria** — Proyecto integrador
