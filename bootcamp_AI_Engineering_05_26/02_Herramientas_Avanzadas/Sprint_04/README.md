# 📘 Sprint 04 — Herramientas Avanzadas: APIs e IA Generativa

## Bienvenido al Sprint 04

En este sprint final de Herramientas Avanzadas aprenderás a integrar tu código Python con el mundo exterior a través de APIs HTTP y a trabajar con modelos de IA generativa. Estas habilidades son fundamentales para construir aplicaciones modernas que interactúen con servicios en la nube y aprovechen el poder de la IA.

Este sprint se divide en **dos unidades temáticas**, cada una con su propia sesión de aprendizaje, ejercicios y práctica obligatoria.

---


## 🎯 Contenidos del Sprint

### 📖 Unidad 01 — HTTP: Llamadas a APIs

**Tema Central**: Integración con APIs REST y manejo de datos en la nube

En esta unidad aprenderás a comunicarte con servidores web a través de HTTP, a consumir APIs públicas y a manejar respuestas en JSON. Estas habilidades te permitirán integrar datos externos en tus aplicaciones Python.

**¿Qué aprenderás?**

- Protocolo HTTP: métodos GET, POST, PUT, DELETE
- Biblioteca `requests` en Python
- Consumo de APIs REST públicas
- Autenticación y manejo de tokens
- Parsing de respuestas JSON
- Manejo de errores y excepciones HTTP
- Rate limiting y buenas prácticas de consumo de APIs
- Construcción de clientes API básicos

**Estructura de la Unidad**:
- 📚 **Teoría** — Material de lectura con ejemplos
- 🎯 **Workout** — Ejercicios guiados para practicar
- 💪 **Ejercicios** — Retos independientes
- 🏆 **Práctica Obligatoria** — Proyecto para evaluar

**Live Review 1**: Lunes 22 de Junio, 19:00–21:00

---

### 🤖 Unidad 02 — IA Generativa con Python: Gemini API

**Tema Central**: Integración de Gemini en aplicaciones Python a través de Google AI Studio

En esta unidad trabajarás con **Gemini**, el modelo de IA generativa de Google, aprendiendo a construir prompts efectivos e integrar esta poderosa herramienta en tus aplicaciones Python. Utilizarás Google AI Studio para experimentar y la Gemini API para integrar IA generativa en tu código.

**¿Qué aprenderás?**

- Gemini API y Google AI Studio: acceso y configuración
- Prompting efectivo con Gemini
- Modelos de lenguaje: características y limitaciones
- Construcción de aplicaciones con Gemini en Python
- Streams y respuestas en tiempo real
- Tokens, costs y optimización de uso
- Casos de uso: clasificación, generación, análisis
- Consideraciones éticas en IA generativa

**Estructura de la Unidad**:
- 📚 **Teoría** — Notebooks con conceptos y ejemplos
- 🎯 **Workout** — Ejercicios guiados en Colab/VSCode
- 💪 **Ejercicios** — Proyectos con IA generativa
- 🏆 **Práctica Obligatoria** — Proyecto integrador

**Live Review 2**: Martes 23 de Junio, 19:00–21:00
