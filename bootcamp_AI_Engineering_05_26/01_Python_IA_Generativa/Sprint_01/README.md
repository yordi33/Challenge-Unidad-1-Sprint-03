# 📘 Sprint 01 — Programación Básica I

## Bienvenido al Sprint 01

En este sprint comenzamos el camino hacia la programación en Python. Consolidaremos las herramientas del Sprint 00 y daremos los primeros pasos en el lenguaje de programación que usarás durante todo el bootcamp y tu carrera como ingeniero de IA.

Este sprint se divide en **dos unidades temáticas**, cada una con su propia sesión de aprendizaje, ejercicios y práctica obligatoria.

---


## 🎯 Contenidos del Sprint

### 📖 Unidad 01 — Introducción a Herramientas

**Tema Central**: Markdown y documentación en el entorno del bootcamp

Esta unidad te introduce a **Markdown**, el lenguaje de marcado que usarás para documentar tu código, escribir READMEs y comunicarte con herramientas de IA. Aunque ya configuraste las herramientas en Sprint 00, ahora aprendes a usarlas de manera efectiva.

**¿Qué aprenderás?**

- Sintaxis completa de Markdown: títulos, énfasis, listas, enlaces, imágenes
- Bloques de código y resaltado de sintaxis
- Tablas, citas, reglas horizontales
- Listas de tareas y otras características de GitHub Flavored Markdown (GFM)
- Cómo estructurar documentos claros y profesionales
- Mejores prácticas para documentación en Jupyter notebooks

**Estructura de la Unidad**:
- 📚 **Teoría** — Material de lectura con ejemplos
- 🎯 **Workout** — Ejercicios guiados para practicar
- 💪 **Ejercicios** — Retos independientes
- 🏆 **Práctica Obligatoria** — Proyecto para evaluar

**Live Review 1**: Lunes 1 de Junio, 19:00–21:00

### 🐍 Unidad 02 — Python I

**Tema Central**: Fundamentos de Python y programación básica

En esta unidad comienzas a escribir código Python. Aprenderás los conceptos fundamentales del lenguaje: variables, tipos de datos, operaciones básicas, impresión de información y estructuras iniciales.

**¿Qué aprenderás?**

- Variables y asignación de valores
- Tipos de datos primitivos: `int`, `float`, `str`, `bool`
- Operaciones aritméticas y lógicas
- La función `print()` y salida de datos
- Conversión de tipos (casting)
- Entrada de datos desde el usuario
- Primeros pasos hacia programas interactivos

**Estructura de la Unidad**:
- 📚 **Teoría** — Notebooks con conceptos y ejemplos (`Basics_I.ipynb`, `Basics_II.ipynb`)
- 🎯 **Workout** — Ejercicios guiados en Colab/VSCode
- 💪 **Ejercicios** — Problemas de programación independientes
- 🏆 **Práctica Obligatoria** — Proyecto integrador

**Live Review 2**: Martes 2 de Junio, 19:00–21:00
