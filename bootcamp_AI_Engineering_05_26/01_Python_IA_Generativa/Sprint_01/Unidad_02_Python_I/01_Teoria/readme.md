# 📘 Sprint 01 · Unidad 02

## Python I — Teoría

En esta unidad se sientan las bases de Python: variables, tipos de datos, operaciones y las primeras colecciones del lenguaje.

---

## 📂 Contenido de la teoria

---

### 📄 1. Python Basics I

🔗 [Abrir notebook](./Basics_I.ipynb)

Primer contacto con Python: declaración de variables, salida por pantalla, comentarios y tipos de datos fundamentales.

**¿Qué aprenderás?**

* Variables y cómo etiquetarlas
* Uso de `print` y comentarios en el código
* Tipos de datos: enteros, flotantes, cadenas y booleanos
* Conversión entre tipos (`int`, `float`, `str`, `bool`)
* Eliminación de variables con `del`

### 📄 2. Python Basics II

🔗 [Abrir notebook](./Basics_II.ipynb)

Continuación de los fundamentos de Python: operaciones con variables y primera introducción a las colecciones mediante las listas.

**¿Qué aprenderás?**

* Operaciones aritméticas y sus propiedades (`+`, `-`, `*`, `/`, `**`, `//`, `%`)
* Operadores de comparación y lógicos
* Funciones y métodos básicos sobre cadenas y listas
* Introducción a las listas: creación, indexado y métodos
