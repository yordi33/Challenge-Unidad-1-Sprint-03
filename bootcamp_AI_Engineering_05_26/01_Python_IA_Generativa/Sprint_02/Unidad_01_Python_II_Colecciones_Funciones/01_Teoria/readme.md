# 📘 Sprint 02 · Unidad 01

## Python II — Colecciones y funciones — Teoría

En esta unidad se trabajan las **colecciones** fundamentales de Python (listas, tuplas, diccionarios y conjuntos) y la definición de **funciones** para organizar y reutilizar código.

---

## 📂 Contenido de la teoria

---

### 📚 1. Colecciones

🔗 [Abrir notebook](./Colecciones.ipynb)

Este notebook introduce listas, tuplas, diccionarios y conjuntos (*sets*), con operaciones habituales, indexado y conversiones entre tipos.

**¿Qué aprenderás?**

* Listas: creación, métodos, ordenación y mutabilidad
* Tuplas y cuándo usarlas frente a listas
* Diccionarios: claves, valores, acceso y modificación
* Conjuntos (*sets*) y operaciones entre conjuntos

---

### ⚙️ 2. Funciones

🔗 [Abrir notebook](./Funciones.ipynb)

Este notebook cubre cómo definir y llamar funciones en Python: argumentos, valores de retorno, ámbito de variables y patrones que aparecen en el cuaderno (por ejemplo recursión, `lambda` o generadores).

**¿Qué aprenderás?**

* Sintaxis de `def`, llamadas y uso de `return`
* Argumentos posicionales, por nombre y variables
* Ámbito de variables dentro y fuera de las funciones
* Recursos avanzados del propio notebook (recursión, `lambda`, generadores, etc.)
