# 📘 Sprint 02 — Programación Básica II

## Bienvenido al Sprint 02

En este sprint profundizamos en Python, trabajando con estructuras de datos más complejas y conceptos avanzados del lenguaje. Aprenderás a trabajar con colecciones, funciones y los fundamentos de la Programación Orientada a Objetos (OOP), herramientas esenciales para escribir código profesional y escalable.

Este sprint se divide en **dos unidades temáticas**, cada una con su propia sesión de aprendizaje, ejercicios y práctica obligatoria.

---


## 🎯 Contenidos del Sprint

### 📖 Unidad 01 — Python II: Colecciones y Funciones

**Tema Central**: Estructuras de datos y funciones para programación efectiva

En esta unidad ampliarás tus habilidades de programación aprendiendo a trabajar con colecciones (listas, tuplas, diccionarios, conjuntos) y funciones. Estas herramientas te permitirán escribir código más modular, reutilizable y eficiente.

**¿Qué aprenderás?**

- Listas, tuplas y sus operaciones
- Diccionarios y manipulación de datos clave-valor
- Conjuntos (sets) y operaciones de conjunto
- Funciones: definición, parámetros y retorno de valores
- Scope y variables globales/locales
- Funciones lambda y comprensiones de listas
- Modularización y reutilización de código

**Estructura de la Unidad**:
- 📚 **Teoría** — Material de lectura con ejemplos
- 🎯 **Workout** — Ejercicios guiados para practicar
- 💪 **Ejercicios** — Retos independientes
- 🏆 **Práctica Obligatoria** — Proyecto para evaluar

**Live Review 1**: Lunes 8 de Junio, 19:00–21:00

### 🐍 Unidad 02 — Python III: Módulos y Programación Orientada a Objetos

**Tema Central**: Introducción a OOP y reutilización de código mediante módulos

En esta unidad entrarás en el mundo de la Programación Orientada a Objetos (OOP), un paradigma fundamental en Python y la industria. Aprenderás a crear clases, objetos y a organizar tu código en módulos reutilizables.

**¿Qué aprenderás?**

- Conceptos fundamentales de OOP: clases, objetos, atributos y métodos
- Constructor `__init__` e inicialización de objetos
- Métodos de instancia y métodos de clase
- Herencia y polimorfismo
- Módulos y paquetes en Python
- Importación de módulos personalizados
- Buenas prácticas en diseño orientado a objetos

**Estructura de la Unidad**:
- 📚 **Teoría** — Notebooks con conceptos y ejemplos
- 🎯 **Workout** — Ejercicios guiados en Colab/VSCode
- 💪 **Ejercicios** — Problemas de programación independientes
- 🏆 **Práctica Obligatoria** — Proyecto integrador
