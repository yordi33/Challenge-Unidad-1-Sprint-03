# Cómo usar IA para estudiar y hacer ejercicios

## Introducción

Durante el bootcamp, ChatGPT y otras herramientas de IA pueden ser un apoyo disponible 24/7. La clave está en **usarlas como apoyo de aprendizaje, no como solucionador instantáneo de problemas**.

El objetivo es que entiendas los conceptos, pero no que copies soluciones. Una IA bien utilizada acelera tu aprendizaje; mal utilizada, te impide aprender.

---

## ❌ Cómo NO usar IA

**Evita estos patrones:**

- **Copiar soluciones directamente**: "Resuelve este ejercicio de CodeWars" → copiar la respuesta y enviarla sin entender
- **Preguntar sin contexto**: "¿Cómo hago un bucle?" sin mostrar lo que ya has intentado
- **Dejar de pensar**: Usar IA para evitar esfuerzo mental, no para complementarlo
- **No verificar la respuesta**: Asumir que lo que dice IA siempre es correcto (a veces miente o comete errores)
- **Saltarse conceptos base**: Usar IA para "saltarte" partes del prework

## ✅ Cómo SÍ usar IA (como tutor)

### 1. **Explicar conceptos que no entiendes**

```
"Acabo de leer sobre list comprehensions en Python.
No entiendo bien la sintaxis. 
¿Puedes explicarme de forma simple qué hace [x*2 for x in range(5)]?"
```

**Por qué funciona**: La IA actúa como profesor explicando un tema, no resolviendo tu problema específico.

### 2. **Pedir pistas en lugar de soluciones**

```
"Estoy haciendo una kata de CodeWars sobre invertir strings.
He intentado usar slicing pero no me sale. 
¿Qué métodos de strings podrían servirme?"
```

**Por qué funciona**: Te da dirección sin darte la respuesta. Tú sigues pensando.

### 3. **Preguntar sobre tu código**

```
"He escrito este código:
[tu código aquí]

No me funciona. ¿Qué está mal? ¿Cómo lo debuguearía?"
```

**Por qué funciona**: La IA te ayuda a entender qué falla, no simplemente lo arregla.

### 4. **Pedir que te haga preguntas**

```
"Quiero entender mejor cómo funcionan los bucles for.
¿Puedes hacerme 3 preguntas para comprobar si lo entiendo?"
```

**Por qué funciona**: El aprendizaje activo es más efectivo. Las preguntas te obligan a pensar.

### 5. **Validar tu entendimiento**

```
"Creo que entiendo cómo funcionan las funciones.
¿Es correcto decir que [tu interpretación]?"
```

**Por qué funciona**: La IA confirma si tu comprensión es correcta o te señala qué está mal.

## 📋 Prompts efectivos para estudiar

### Para entender un concepto

```
Explícame qué es [concepto] de forma simple, como si tuviera 10 años.
Después dame un ejemplo práctico en Python.
```

### Para un ejercicio atascado

```
Tengo este ejercicio:
[descripción del ejercicio]

He intentado esto:
[tu código]

Pero no funciona porque [lo que crees que es el problema].
¿Qué estoy pasando por alto? No me des soluciones directas, ayúdame a llegar a ellas.
```

### Para revisar tu solución

```
¿Es esta una buena forma de resolver [problema]?
[tu código]

¿Hay formas mejores o más eficientes?
¿Cómo mejoraría la legibilidad?
```

### Para practicar

```
Dame 3 ejercicios para practicar [concepto].
No quiero soluciones, solo descripciones de lo que debo hacer.
Después te mostraré mis intentos.
```

## 🎯 El "Prompt Maestro"

**Usa esto al inicio de cada conversación con la IA para instruirla sobre cómo ayudarte:**

```
Eres mi tutor de programación durante un bootcamp. Tu rol es GUIARME 
para que llegue a las soluciones por mí mismo, NUNCA darme soluciones directas.

REGLAS:
1. NUNCA escribas código completo que resuelva mi problema
2. Cuando no entienda un concepto, explícamelo con ejemplos simples
3. Cuando me atasque en un ejercicio, hazme preguntas para guiarme hacia la respuesta
4. Sugiere conceptos que podrían servir, pero no me digas exactamente cómo usarlos
5. Si te pido la solución, recházala y reoriéntame hacia el aprendizaje
6. Ayúdame a debuguear explicándome qué está mal, no arreglándolo por mí
7. Después de que resuelva algo, pregúntame si lo entiendo y pídeme que lo explique

```

**Cópialo, pégalo en tu chat con la IA al empezar**, y desde ese momento actuará como se espera. Funciona con ChatGPT, Claude y la mayoría de IA de conversación.

---

## ⚡ Regla de oro: LA IA NUNCA TE DA SOLUCIONES

**La IA nunca debe darte código completo que resuelva tu problema. Si lo hace, es porque no la instruíste bien.**

**Si la IA insiste en darte una solución completa:**
- Repite el Prompt Maestro
- Remítele a que tu objetivo es aprender, no copiar
- Cierra la conversación y empieza una nueva

El aprendizaje no viene de tener la respuesta. Viene de luchar por encontrarla.
